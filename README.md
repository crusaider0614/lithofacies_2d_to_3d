# Lithofacies Prediction 2D to 3D

This repository is for sharing source codes of the second stage in the paper entitled "Integrated Rock Physics, Seismic, and Machine Learning Methodology for Regional 3D Lithofacies Modeling-A Case Study from the Jeju Basin, Offshore Korea."

Some large files—trained models and curated subdataset for testing—will be provided as a public download link.
